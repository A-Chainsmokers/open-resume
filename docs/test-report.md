## 发现的关键问题

### 1. 设置弹窗被遮罩层阻挡（严重）
- **现象**：点击设置按钮后，设置弹窗元素在 DOM snapshot 中不可见或 click 事件被拦截
- **影响范围**：模板切换、深色主题、模块排序、AI 配置 UI、PDF/JSON 导出按钮等全部设置相关功能
- **可能原因**：Ant Design Modal 的遮罩层（`.ant-modal-mask`）或 portal 层问题

### 2. AI 后端接口返回 400 错误（严重）
- `/api/ai/polish`：无论是否填写 API Key，均返回 400
- `/api/ai/models`：返回 502 Bad Gateway（后端尝试连接配置的 baseUrl 失败）
- **影响**：AI 润色、AI 聊天修改功能完全不可用

### 3. 预览模式切换行为异常
- 点击 segmented control 的"预览"后，编辑器侧边栏仍然可见，与"隐藏编辑器全屏显示简历"的预期不符

### 4. 自动化测试限制
- 浏览器文件选择对话框无法自动化，影响 JSON 导入、PDF/图片上传识别等测试
- 缺少 resize 命令导致响应式布局测试无法进行
